# TexHolding MVP P0 Frontend

基于 `/mvp-p0` Canonical Package 启动的前端开发骨架。

## 当前范围

- Welcome
- Onboarding Step 1 / Step 2
- Training
- Training Feedback Sheet
- Training Result
- Reminder
- Login Sheet
- Home
- Review Input
- Review Result

## 当前实现

- `React + Vite + TypeScript` 工程骨架
- P0 主题与基础样式
- 路由与页面骨架
- 共享组件
- 本地状态持久化
- mock runtime 数据
- 训练结果 / 提醒 / 复盘结果的基础规则逻辑

## 运行

项目依赖 Node.js 环境。

```bash
npm install
npm run dev
```
